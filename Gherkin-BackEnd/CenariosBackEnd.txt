
1 - Criar usuário com sucesso
2 - Consultar usuário criado
3 - Alterar cadastro do usuário criado
4 - Deletar usuário criado
5 - Consultar usuário deletado


1)
Quando enviar uma requisição POST para a URL https://dummy.restapiexample.com/api/v1/create
E informar o Body com os valores <name> <salary> <age>
|name||salary||age|
|Afonso||5000||20|
Entao a API deve retornar código Http 200
E o atributo data no responseBody não deve estar vazio


2)
Quando enviar um requisição GET para a URL https://dummy.restapiexample.com/api/v1/employee/10
Entao a API deve retornar código Http 200
E o atributo data no responseBody deve conter <id> <employee_name> <employee_salary> <employee_age>
 |id| |employee_name| |employee_salary| |employee_age|
 |10| |Sonya Frost  | |103600         | |23          |
 
 
3)
Quando enviar uma requisição PUT para a URL https://dummy.restapiexample.com/api/v1/update/12
E informar o Body com os valores <name> <salary> <age>
|name||salary||age|
|Gustavo||1200||40|
Entao a API deve retornar código Http 200
E o atributo message no responseBody deve conter o texto "Successfully! Record has been updated."


4)
Quando enviar uma requisição DELETE para a URL https://dummy.restapiexample.com/api/v1/delete/9
Entao a API deve retornar código Http 200
E o atributo message no responseBody deve conter o texto "Successfully! Record has been deleted"


5)
Quando enviar uma requisição GET para a URL https://dummy.restapiexample.com/api/v1/delete/5
Entao a API deve retornar código Http 404 Not Found
E o atributo message no responseBody deve conter o texto "Error Occured! Page Not found, contact rstapi2example@gmail.com"
